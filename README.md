## 📄 Sobre o Projeto

O objetivo principal era substituir o agendamento manual por um sistema automatizado, intuitivo e que aumentasse a conversão de clientes via WhatsApp.

Este projeto foca na experiência do usuário (UX), garantindo que o cliente consiga agendar um serviço em poucos cliques, tanto no mobile quanto no desktop.

## ✨ Funcionalidades Principais

* **Agendamento Intuitivo:** Interface limpa para seleção de serviços, datas e horários disponíveis.
* **Integração com WhatsApp:** Ao finalizar o agendamento, o sistema gera uma mensagem automática formatada e encaminha o cliente diretamente para o WhatsApp da empresa para confirmação imediata.
* **Painel Administrativo (Dashboard):** Interface visual para que o dono da estética possa visualizar os agendamentos do dia/mês (Front-end visualizado).
* **Design Responsivo:** Layout totalmente adaptado para celulares, tablets e desktops.
* **Feedback Visual:** Animações e transições suaves para melhorar a interatividade.
